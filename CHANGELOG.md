# Changelog

История заметных изменений проекта Говорун.
Самые свежие версии — сверху.

## 1.0.4 — 26.04.2026

### Новые возможности

- **Удержание для записи фразы** — зажмите птичку, говорите, отпустите для отправки текста. Длинная фраза распознаётся целиком, без разбиений на паузах.
- **Настройка внешнего вида птички** — выбор размера и стороны экрана (слева/справа). Живой превью прямо в Настройках при изменении.
- **Кнопка «Сбросить настройки»** — возвращает размер, прозрачность, сторону и положение птички к дефолту в один тап.
- **Лёгкая вибрация по умолчанию** — короткая обратная связь при старте и окончании записи. Можно отключить в Настройках.
- **Карточка «Что нового»** на главном экране после обновления.

### Исправления

- Птичка теперь **запоминает позицию после перетаскивания** — больше не сбрасывается в центр после перезапуска приложения или сервиса.
- Свайпы через птичку (например, системный жест «назад») больше не активируют запись.
- Птичка не съезжает за пальцем во время удержания.
- При недоступном микрофоне открывается главный экран с инструкцией как восстановить разрешение.
- Защита от краша при занятом микрофоне (другое приложение, отозванное разрешение).
- Несколько мелких улучшений интерфейса для более плавной работы.

### Под капотом

- Корректное освобождение модели распознавания при нехватке памяти у системы.
- Защита от гонки при одновременном освобождении и использовании модели.
- Использование `View.performHapticFeedback` для вибрации, обходящее системную галочку «Touch feedback».
- Реакция на сигнал `TRIM_MEMORY_RUNNING_LOW` для лучшего поведения на устройствах с малой ОЗУ.
